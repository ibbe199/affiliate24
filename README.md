# Affiliate24

Affiliate24 is a free GitHub Pages affiliate website starter project.

This project is designed to help beginners launch an affiliate marketing site with zero hosting cost using static HTML and CSS.

## Features

- Lightweight HTML structure
- 5 ready-made review pages
- SEO-friendly layout
- GitHub Pages compatible
- Easy to customize

## Pages Included

- index.html
- review-chatgpt.html
- review-copyai.html
- review-japer.html
- review-notionai.html
- review-surfer.html
